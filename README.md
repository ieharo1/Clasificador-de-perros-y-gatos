# 🐕 Clasificador de Perros y Gatos

Proyecto de machine learning para clasificación de imágenes de perros y gatos desarrollado por **Isaac Esteban Haro Torres**.

---

## 📝 Descripción

Modelo de machine learning para clasificación automática de imágenes, distinguiendo entre perros y gatos. Utiliza técnicas de deep learning y redes neuronales convolucionales.

---

## ✨ Características

- Clasificación de imágenes
- Modelo de deep learning
- Entrenamiento con dataset de perros y gatos
- Predicción en tiempo real

---

## 🛠️ Stack Tecnológico

- Python
- TensorFlow / Keras
- OpenCV
- NumPy / Pandas

---

## 🚀 Uso

```bash
# Entrenar modelo
python train.py

# Predicción
python predict.py --image ruta/imagen.jpg
```
